Dungeon Game (C#).
Консольная игра в жанре dungeon crawler.

Логика программы.
Проект реализован с использованием объектно-ориентированного программирования.

В программе реализованы:
- базовый класс Character.
- класс игрока Player.
- классы противников Enemy, Goblin, Orc, Skeleton.
- система оружия Weapon.
- система предметов Potion.
- механика боя.
- взаимодействие объектов через методы классов.
- управление игрой через консоль.

Технологии использованные при написании кода:C#, ООП.
